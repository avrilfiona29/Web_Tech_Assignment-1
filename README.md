# Resume & Bio-Data Website

A professional multi-page website showcasing resume and bio-data with modern styling and interactive features.

## Features

- **Multiple Pages**: Home, Resume, Bio-Data, and Contact pages
- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Modern Styling**: Clean CSS with animations and transitions
- **Interactive Elements**: Form validation, smooth scrolling, hover effects
- **Professional Layout**: Well-organized content presentation

## Files Structure

```
├── index.html          # Home page
├── resume.html         # Resume page
├── biodata.html        # Bio-data page
├── contact.html        # Contact page
├── styles.css          # All styling
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Technologies Used

- HTML5
- CSS3 (with animations and flexbox/grid)
- JavaScript (vanilla JS for interactivity)
- Responsive design principles

## How to Use

1. **Customize Content**: Replace placeholder information with your actual details
2. **Test Locally**: Open `index.html` in a web browser
3. **Deploy**: Upload to any cloud hosting platform

## Cloud Hosting Options

### Option 1: GitHub Pages (Free)
1. Create a GitHub repository
2. Upload all files
3. Go to Settings > Pages
4. Select main branch and save
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify (Free)
1. Sign up at netlify.com
2. Drag and drop your project folder
3. Site will be live instantly with a custom URL

### Option 3: Vercel (Free)
1. Sign up at vercel.com
2. Import your project
3. Deploy with one click

### Option 4: Firebase Hosting (Free)
```bash
npm install -g firebase-tools
firebase login
firebase init hosting
firebase deploy
```

## Customization Guide

1. **Personal Information**: Update name, contact details, and bio in all HTML files
2. **Colors**: Modify CSS variables in `styles.css` (`:root` section)
3. **Content**: Add your education, experience, skills, and projects
4. **Images**: Add profile photo by creating an `images` folder and updating HTML
